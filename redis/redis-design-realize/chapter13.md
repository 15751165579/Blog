# 服务器

### 命令请求的过程

  执行一条Redis命令的大致流程为：

  - 1）客户端将命令转化为协议格式，发送给服务器。
  - 2) 服务器将命令放入输入缓存区，接下来解析命令中的参数。
  - 3）服务器对命令进行处理，并生成命令的回复，放入输出缓冲区。
  - 4）客户端转化协议格式，呈现给用户。

### serverCron

  前面以及介绍了serverCron属于Redis中的周期时间事件，负责很多功能，如：内存占用的情况，清理失效的客户端连接，处理过期键值对等等。

### 初始化服务器

  Redis服务器启动的过程分为5步：

  - 1）初始化服务器状态。
  - 2）载入服务器配置文件。
  - 3）初始化服务器数据结构。
  - 4）载入RDB文件（也可能载入AOF文件）。
  - 5）启动事件循环。
