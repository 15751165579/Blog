# JavaScript HISTORY

## 发展的历程

#### 1995年9月，Netscape推出了LiveScript。

> 这是Brendan Eich仅花了10天时间完成的作品。

#### 同年12月，更名为JavaScript。

> 蹭了一波Java的热度。

#### 实际上在1995年12月，Server-side JavaScript已经出现。

#### 1996年，Netscape将JavaScript提交给了Ecma International进行标准化，并推出新的语言标准ECMAScript，而JavaScript就是ECMAScript标准的著名实现，ActionScript和JScript也是比较有名的实现。

> ActionScript是为Flash产品开发的编程语言，只不过它的文档对象模型以SWF格式动画为主。JScipt则是微软开发的一款编程语言。

#### 自从开启ECMAScript标准制定后，似乎并没有什么大的变化，自从1999年推出ES3之后，十年内多没有任何的改动。但是各大厂商却在争先恐后的拓展自己语言。

> 10年无变化的原因之一：ES4过于激进，并没有并入标准。而唯一实现ES4的就是ActionScript。

#### 2009年发布ES5，严格模式开始。

#### 2012年，开发者们才推动使用ES5风格来编写代码。

#### 2015年，发布ES6（也叫ES2015），并且负责制定ECMAScript规范草案的委员会（TC39）决定将定义新标准的制度改为1年1次，配合常青浏览器每6周发布一次新版的快速迭代日程，加速JavaScript/ECMAScript的进化。
